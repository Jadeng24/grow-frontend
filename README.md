# Grow.com Front-end

## Coding Challenge

#### Instructions

1. `git clone https://github.com/Jadeng24/grow-frontend.git`

2. cd into `grow-frontend`

3. `npm install`

4. `npm start`

5. project will be running at `http://localhost:3002/`

6. select "Representative" or "Senator"

7. select "state"

8. view person details
